<p>
  O projeto Fokus inclui um temporizador estilo Pomodoro e uma lista de tarefas. Ele foi criado durante o curso de JavaScript da Alura focado na manipulação de elementos do Dom.

  Desta vez, a estrutura do HTML e CSS já estavam prontas e o objetivo era construir uma lógica que pusesse a aplicação em ação, tornando-a dinâmica.
  
  Foi usado innerHTML para alterar textos, setInterval e clarInterval para executar e interromper a execução de trechos do código, e objetos de áudio, explorando suas propriedades e métodos.
  
  Também foi implementado o objeto Date para formatar minutos e segundos, configurado diferentes temporizadores para três contextos diferentes (foco, descanso curto e descanso longo), e foi inserido temporizador na tela.
  
  O localStorage foi utilizado para armazenar e recuperar dados da lista de tarefas, permitindo persistência entre sessões; elementos foram criados com o document.createElement e aninhados utilizando o método append.
  
  Os elementos HTML foram selecionados através do querySelector, a interação e manipulação de classes feitas com classList.toggle; eventos criados e adicionados com addEventListener; comportamentos padrão, como o recarregamento da página ao submeter um formulário, prevenidos com preventDefault.
</p>
